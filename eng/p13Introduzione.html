Moving Towards a Smart Grid

Electricity, once produced, travels through the grid. These grids can be viewed like “electricity roads” and traditionally have been one-way in order to produce an uninterrupted distribution of a large and constant flow of energy, generated by a few power stations, to a great number of consumers.
Today, however, the world has changed. For example, power plants which use renewable sources (photovoltaic and wind sources) are continuously increasing and feature non-continuous production: the blade of a wind turbine functions only when there is wind, and photovoltaic panels need sunshine!
Thus, smart grids have been invented, which are both communication networks and active, bidirectional electrical grids, which allow for the interaction between the producers and consumers, processing and monitoring an enormous quantity of information in real time.
